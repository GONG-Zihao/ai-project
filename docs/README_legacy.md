# AI个性化学习助手

## 项目简介
本项目是一个基于大语言模型的AI个性化学习助手，支持文本/图片题目输入、AI自动答疑、错题本管理、个性化学习建议和学习进度追踪。适合学生和自学者提升学习效率。

## 主要功能
- 文本/图片题目输入与AI答疑
- 错题本自动归档与回顾
- 个性化学习计划与资源推荐
- 学习进度可视化追踪

## 运行方法
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动应用：
   ```bash
   streamlit run app.py
   ```

## 依赖说明
- streamlit：前端界面与交互
- deepseek：AI大模型API
- pillow：图片处理
- pandas：数据管理

## 创新亮点
- AI原生、全流程体验
- 支持多模态输入
- 个性化推荐与错题管理
